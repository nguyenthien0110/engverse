# EngVerse

EngVerse is a simple English learning web app built with Next.js.

## Live Website

Visit the app here:

https://engverse-eta.vercel.app/

## Features

- Create and manage topics
- View topic details
- Store data in the browser
- Fast and simple interface

## Technology

- Next.js
- TypeScript
- React
- IndexedDB

## Run Locally

npm install
npm run dev

## Build

npm run build

## Notes

- Data is stored in your browser.
- If you clear browser data, your content will be lost.
- No backend server is used.
